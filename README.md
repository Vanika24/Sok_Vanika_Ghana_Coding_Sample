 ## Stata Coding Sample
 **Author:** Vanika Sok

  ### Overview
  This repository contains Stata code for cleaning and analyzing survey data
  from Ghana, including household assets, demographics, and mental health
  outcomes.

  ### File Structure
  - `00_master_ghana.do` — master script that runs all files in order
  - `01_cleaning.do` — data cleaning and variable construction
  - `02_part2q1.do` — descriptive statistics and balance tables
  - `03_part2q2q5.do` — regression analysis and treatment effect estimation

  ### Software
  Stata (version 17+)
